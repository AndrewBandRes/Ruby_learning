Детская игра "Виселица"
Необходимо по буквам отгадать загаданное слово
Если ошибиться 7 раз, то игра будет проиграна.
Чтобы начать, используйте команду

ruby hangman.rb <загаданное слово>

Аргумент не является обязательным, список слов
можно разместить в файле words.txt в папке data

Особенности правил:
- повторы букв не считаются ошибкой
- буквы "е", "ё", а также "и", "й" считаются "друзьями"
и открывают друг друга

Написана с использованием:
ruby 2.7.1p83 (2020-03-31 revision a0c7c23c9c) [x86_64-linux]

Необходима библиотека unicode_utils.
Для установки воспользуйтесь командой

gem install unicode_utils
